rbgo compositor

  • Drag titlebar — move a window
  • Drag bottom-right corner — resize
  • Click a window — focus & raise
  • × in a titlebar — close
  • Right-click desktop — root menu (spawn)
  • Right-click a window — raise / close
  • Tab — cycle focus · Esc — dismiss menu
Wayland-inspired single-instance compositor + Openbox-style WM policy, written in pure Ruby, running in WebAssembly.
