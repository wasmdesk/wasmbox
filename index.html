rbgo compositor

  • Drag titlebar — move a window
  • Drag bottom-right corner — resize
  • Click a window — focus & raise
  • × in a titlebar — close
  • Right-click desktop — root menu (spawn)
  • Right-click a window — raise / close
  • Tab — cycle focus · Esc — dismiss menu
Wayland-inspired single-instance compositor + Openbox-style WM policy, written in pure Ruby, running in WebAssembly. Step C: the compositor itself runs in a dedicated Web Worker; external clients run in their own Web Workers and talk to the compositor worker directly via postMessage + SharedArrayBuffer.
